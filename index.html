📖 How to Use These Results

  • For Repeatability:
    • Repeatability limit (r): Maximum difference between two results under same conditions
    • Use for quality control within the same lab
  • For Reproducibility:
    • Reproducibility limit (R): Maximum difference between two results from different labs
    • Use for comparing results between laboratories
  • For Uncertainty:
    • Use the expanded uncertainty (U) for all test reports
    • Report as: MDD = X.XXX ± U.XXX g/cm³ (k=X.X, 95% confidence)
